# 통근 패턴 분석 — 시간대별 유입·유출의 거울상 (교통카드 실측)

> **분석 질문에 대한 가장 직접적인 실측 답:** "왜 판교는 성공하고 청라는 실패했는가"를
> 등시간권(도달 *가능* 노동시장, 이론)·직주비(거주지 기준 통계)가 아니라
> **실제로 사람이 언제·어느 방향으로 오갔는가**(교통카드 실측 통행)로 보인다.
> 결과는 두 지역이 **완벽한 거울상**(mirror image)이라는 것이다 — 판교는 *일자리 도시*, 청라는 *베드타운*.

## 0. 범위·단위 4종 (§5-A 의무 명시)

| 항목 | 내용 |
|---|---|
| **시간범위** | 2024-11-14(목) 1일, 00~23시 |
| **공간범위** | 판교 구역(분당구 삼평동, 2.84㎢) · 청라 구역(서구 청라1·2·3동, 20.53㎢) |
| **공간단위** | 구역 경계 내 대중교통 정류장(지하철역+버스정류장) 승하차 기준 → 구역 단위 집계 |
| **시간단위** | 1시간(24개 구간), 카드 태그 시각 기준 |
| **출처·기준월** | 교통카드 통행사슬 `smartcard_tripchains_seoul_20241114.parquet` (2024-11) |
| **산출** | `03_analysis/scripts/commute_pattern.py` → `04_system/web/public/data/commute_hourly.json` |

**정의** — 구역을 기준으로:
- **유입(in)** = 마지막 하차역이 구역 내 (= 사람들이 이 구역으로 *도착*: 출근·방문)
- **유출(out)** = 첫 승차역이 구역 내 (= 사람들이 이 구역에서 *출발*: 퇴근·외출)

따라서 *아침 유입*=출근 도착지, *아침 유출*=거주지에서 나감, *저녁 유입*=귀가 도착, *저녁 유출*=퇴근 출발.

## 1. 핵심 결과 — 한 장의 거울상

| 지역 | 유입 피크 | 유출 피크 | 패턴 해석 |
|---|---|---|---|
| **판교** | **오전 7시** (7,227명) | 저녁 18시 (5,142명) | 아침에 모이고 저녁에 빠짐 → **일자리 도시(직장 유입형)** |
| **청라** | 저녁 18시 (2,745명) | **오전 7시** (2,907명) | 아침에 빠지고 저녁에 돌아옴 → **베드타운(주거 유출형)** |

두 지역의 유입·유출 피크 시각이 **정확히 뒤바뀐다**(판교 유입 7시 = 청라 유출 7시 / 판교 유출 18시 = 청라 유입 18시).
이는 직주비(판교 4.14 vs 청라 0.28)·등시간권(도달종사자 13.9배)이 가리키는 "판교=고용중심, 청라=주거중심"을
**거주지 통계가 아닌 실제 통행의 시간 구조로 독립 확증**한다.

## 3. 첨두 집계 — 비율이 결정적 (표 2)

| 구간 | 지표 | 판교 | 청라 |
|---|---|---:|---:|
| **오전 첨두 07–09시** | 유입 | 17,305 | 4,049 |
| | 유출 | 2,300 | 6,923 |
| | **유입/유출 비** | **7.52** | **0.58** |
| | 순(유입−유출) | **+15,005** | −2,874 |
| **저녁 첨두 17–19시** | 유입 | 2,702 | 6,718 |
| | 유출 | 13,590 | 5,190 |
| | **유입/유출 비** | **0.20** | **1.29** |
| | 순(유입−유출) | −10,888 | **+1,528** |

**해석의 핵심:**
- 판교는 **오전에 유입이 유출의 7.5배**, 저녁에 **유출이 유입의 5배**(=유입/유출 0.20). 출근하러 들어오고 퇴근하러 나가는 전형적 업무지구.
- 청라는 정반대로 **오전에 유출이 유입의 1.7배**(=0.58의 역수), 저녁에 **유입이 유출의 1.3배**. 아침에 나가고 저녁에 들어오는 전형적 주거지.
- 두 지역의 오전 유입/유출 비(7.52 ↔ 0.58)는 **약 13배 차이** — 직주비 격차(4.14 vs 0.28 ≈ 15배), 등시간권 도달종사자 격차(13.9배)와 **같은 자릿수**로 수렴한다. 서로 다른 데이터(거주지 통계·지하철 그래프·교통카드)가 한 결론을 가리킨다.

## 5. 그래프(시스템 차트) 분석 포인트 — 무엇을 어떻게 보여줄까

> 지도 위 화살표(OD arc) 대신 **24시간 유입·유출 라인차트**로 제시(이동 자체보다 *시간 구조*가 메시지). 두 지역을 한 화면에서 비교.

**권장 차트 구성:**
1. **2지역 × 2계열 라인차트** — x축 00~23시, y축 통행 수. 계열 4개: 판교 유입(파랑 실선)/판교 유출(파랑 점선)/청라 유입(주황 실선)/청라 유출(주황 점선). 한 패널에 겹쳐 거울상을 직관적으로.
   - 대안: 좌(판교)·우(청라) 2패널 side-by-side로 각 지역의 유입·유출 교차를 또렷이.
2. **순유입(net) 막대/영역** 보조 패널 — 판교는 오전 양(+)·저녁 음(−), 청라는 오전 음(−)·저녁 양(+). 거울상이 한눈에.

**차트에 달 주석(annotation) — 캡션 후보:**
- 판교 7시 유입 정점에 "**오전 7시 유입 정점 7,227명 — 출근 도착**" 콜아웃.
- 청라 7시 유출 정점에 "**오전 7시 유출 정점 2,907명 — 거주지에서 출발**" 콜아웃.
- 헤드라인: "**판교는 아침에 채워지고(유입/유출 7.5배), 청라는 아침에 비워진다(0.58배)**".

**차트 하단 고정 캡션(범위·단위 4종 포함):**
> 교통카드 통행사슬 기준 시간대별 유입(목적지=구역)·유출(출발지=구역). 단위 통행 수(건), 대중교통 한정. 공간단위 구역 내 정류장 승하차. 2024-11-14(목) 1일. 출처: 교통카드 통행사슬 데이터.

## 6. 한계 (보고서 §5 반영)

- **대중교통 한정**: 교통카드 통행만 집계되어 **승용차 통근은 제외**. 판교는 승용차 통근 비중이 커 절대 통행량은 과소집계일 수 있다. 다만 본 분석의 결론은 절대량이 아니라 **유입·유출의 시간 구조(거울상)** 이며, 이는 표본 수단에 강건하다.
- **1일 단면(목요일)**: 2024-11-14 하루 스냅숏. 요일·계절 변동 미반영(평일 통근의 전형적 단면으로 해석).
- **정류장 PIP 경계효과**: 구역 경계 인근 정류장의 귀속에 따라 소폭 변동 가능(구역계는 행정동 경계로 고정).
- **기준월 상이(가점 보조지표)**: 교통카드 2024-11 / 통신사 2025-02 / 지정데이터 2023·현행. **핵심 비교(필수지표)는 동일기준**, 본 통근 패턴은 출처·기준월 명기 후 **보조지표**로 활용.
- **상관≠인과**: 통근 패턴은 "판교=고용중심, 청라=주거중심"이라는 *상태*를 실측할 뿐, 그 원인(접근성·정책·시기)의 인과는 §5 한계에서 절제한다.
